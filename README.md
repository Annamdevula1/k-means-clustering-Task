# k-means-clustering-Task
# Customer Segmentation using K-Means Clustering

## Objective
Group customers into clusters based on income and spending behavior.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
Mall_Customers.csv Dataset

## Steps
1. Load dataset
2. Select features
3. Use Elbow Method
4. Apply K-Means clustering
5. Generate cluster labels
6. Visualize clusters and centroids

## Result
Successfully segmented customers into different groups using K-Means clustering.
